Debian9.2 安装 MariaDB

发表于 2018-08-07

安装MariaDB：

更改配置，允许远程访问：

  1. 打开 /etc/mysql/mariadb.conf.d/50-server.cnf
  2. 找到 bind-address = 127.0.0.1 并注释

添加远程登录帐号（Shell下）

  1. 使用 root 权限运行

  2. 进入mysql系统数据库

  3. 配置所有ip 可以通过帐号root 密码123 访问数据库

  4. 从mysql数据库中的授权表重新载入权限

  5. 退出MariaDB 重启系统or重启MariaDB
